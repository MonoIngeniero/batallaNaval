# 🚢 Batalla Naval en Java

## Descripción

Proyecto desarrollado como parte del curso de Algoritmos y programación II de la Universidad Icesi.

Consiste en una implementación del clásico juego **Batalla Naval**, donde un jugador compite contra la computadora en un tablero de 10x10. El proyecto fue diseñado aplicando principios de programación orientada a objetos, modularidad y documentación del código mediante JavaDoc.

---

## Tecnologías

- Java
- Programación Orientada a Objetos (POO)
- JavaDoc

---

## Funcionalidades

- Modo de juego contra la computadora.
- Tablero de 10x10.
- Modo estándar y modo personalizado.
- Colocación de barcos.
- Sistema de disparos por turnos.
- Verificación de impactos y barcos hundidos.
- Condición de victoria automática.
- Validación de coordenadas y disparos repetidos.
